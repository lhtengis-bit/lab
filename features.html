Features

One keyboard, a whole toolbox

Hall-effect switches make these features possible. Use the tabs to explore what each one actually does.

Rapid Trigger

The feature Wooting pioneered, and the one that changed the peripheral landscape. Instead of waiting for a key to travel back past a fixed reset point, Rapid Trigger resets and re-activates a key the instant you change direction.

  • Repeat inputs in rapid succession with no perceived delay
  • Razor-sharp counter-strafing in tactical shooters
  • Sensitivity is fully adjustable per key in Wootility
0 ms
perceived reset delay

Adjustable Actuation

You decide how far a key must travel before it registers. Set a hair-trigger 0.1 mm for movement keys and a deliberate 4.0 mm for keys you don't want to fire by accident — all configurable per key.

  • 0.1 mm precision across the full travel range
  • Quick presets: 0.1, 0.5, 1, 2, 3 or 4 mm
  • Different actuation points for different keys
0.1–4.0
mm actuation range

The Power of Analog

An analog keypress isn't binary — the keyboard tracks exactly how far you press. Activate analog movement and your WASD keys gain variable input, so you can walk, jog or sprint at precisely the speed you choose.

  • Variable movement like an analog stick
  • Feather the throttle and brake in racing games
  • Backed by Wooting's open Analog SDK
∞
degrees of input, not just on/off

Rappy Snappy & Snappy Tappy (SOCD)

Rappy Snappy lets you choose two keys that are never active at the same time — the key pressed deeper stays active. Snappy Tappy (SOCD) is similar, but lets you decide how the winner is chosen, for example prioritising the key pressed last.

  • Clean opposing-direction inputs (A vs D)
  • Choose "deeper wins" or "last input wins"
  • Bottom out both keys to override
A ⇄ D
no conflicting inputs

DKS, Mod-Tap & Toggle

Dynamic Keystroke (DKS) maps multiple actions to a single key — one on a half-press, another on a full press, and more on release. Add Mod-Tap (tap for A, hold for B) and Toggle keys, and a single key can do a lot.

  • Up to four actions bound to one key with DKS
  • Mod-Tap adds a second action to any key
  • Toggle a held action, like crouch, with one tap
4-in-1
actions per key with DKS
